## G. 한계 (정직)
1. **정답지 편향**: expected_relevant 가 레시피 이름 부분문자열 기반 → 쿼리에 음식 단어가 있으면 TF-IDF에 유리. 전체 평균이 TF-IDF 우위로 보이는 주 원인. 정성 추천 품질(스테이크/찌개 사례)은 의미가 우세 — 독립 블라인드 재라벨링 필요(NL_RELABELING_PROTOCOL.md).
2. **표본 n=30** → 탐색적. effect size·p값 병기.
3. 잔여 미도달 쿼리는 맵기/난이도/조리시간 필터 또는 코퍼스 이름 불일치 — retrieval_k 확대 여지.
4. 운영(512MB)은 메모리상 TF-IDF 기본, 의미/하이브리드는 시연·평가 또는 경량화 후.
